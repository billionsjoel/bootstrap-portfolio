# My Portfolio Project

> A project that shows my portfolio, technologies and links to the projects i have
designed or worked on using bootrap.

![screenshot](./screen-app.png)

## Built With

- HTML and CSS.
- Bootstrap Framework.
- Git, Github and VsCode.

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- A modern upto-date browser
- Text Editor or IDE

### Setup
 - copy the git link for cloning.

### Install
 - Open terminal
 - run git clone (link)

### Usage
- double click on the html index file to open in browser.
